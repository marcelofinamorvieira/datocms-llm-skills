## Video Upload Tracks

If the asset linked to an upload is a video file (e.g., hosted on Mux), you can add subtitle and caption tracks to it. Tracks are managed through the `client.uploadTracks` resource.

### Track Create Attributes

| Attribute | Type | Description |
|---|---|---|
| `url_or_upload_request_id` | `string` | URL to the subtitle file (SRT/VTT) or an upload request ID |
| `type` | `"subtitles" \| "audio"` | The type of track |
| `name` | `string` (optional) | Human-readable track name (e.g., "English Subtitles") |
| `language_code` | `string` | BCP 47 language code (e.g., `"en"`, `"fr"`) |
| `closed_captions` | `boolean \| null` (optional) | Whether this track provides subtitles for the Deaf or Hard-of-hearing (SDH) |

### Track Response Attributes

| Attribute | Type | Description |
|---|---|---|
| `id` | `string` | Unique track identifier |
| `type` | `"subtitles" \| "audio"` | The type of track |
| `name` | `string` | Human-readable track name |
| `language_code` | `string` | BCP 47 language code |
| `status` | `"preparing" \| "ready" \| "errored"` | Current processing status |
| `error` | `string \| null` | Error explanation when status is `"errored"` |
| `closed_captions` | `boolean \| null` | Whether this track provides SDH subtitles |
| `upload` | `{ type: "upload", id: string }` | Reference to the parent upload |

## Creating a Track

Add a subtitle track to a video upload by providing a URL to an SRT or VTT file:

```ts
const englishSubtitleTrack = await client.uploadTracks.create("upload-id", {
  type: "subtitles",
  url_or_upload_request_id: "https://example.com/subtitles/en.vtt",
  name: "English Subtitles",
  language_code: "en",
  closed_captions: false,
});

console.log(englishSubtitleTrack.id, englishSubtitleTrack.status);
```

**Important:** Track creation is an async job. The returned track may have a `status` of `"preparing"` while the file is being processed.

### Creating a Closed Captions Track

```ts
const closedCaptionTrack = await client.uploadTracks.create("upload-id", {
  type: "subtitles",
  url_or_upload_request_id: "https://example.com/subtitles/en-cc.vtt",
  name: "English (CC)",
  language_code: "en",
  closed_captions: true,
});
```

## Listing Tracks

Retrieve all tracks associated with a video upload:

```ts
const tracks = await client.uploadTracks.list("upload-id");

for (const track of tracks) {
  console.log(track.id, track.name, track.language_code, track.status);
}
```

### Filtering by Status

```ts
const tracks = await client.uploadTracks.list("upload-id");

const readyTracks = tracks.filter((track) => track.status === "ready");
const erroredTracks = tracks.filter((track) => track.status === "errored");

for (const erroredTrack of erroredTracks) {
  console.log(`Track "${erroredTrack.name}" failed: ${erroredTrack.error}`);
}
```

## Deleting a Track

Remove a track from a video upload by providing both the upload ID and the track ID:

```ts
const deletedTrack = await client.uploadTracks.destroy("upload-id", "track-id");

console.log("Deleted track:", deletedTrack.name);
```

**Important:** The `generateSubtitles()` method is deprecated and should not be used in new code.

## Upload Tags (Manual)

Manual tags let you organize assets in the media library. These are tags explicitly created by users, as opposed to smart tags generated by AI.

### Listing Manual Tags

Tags are sorted by name and paginated (defaults to 50, maximum 500 per page):

```ts
const manualTags = await client.uploadTags.list();

for (const tag of manualTags) {
  console.log(tag.id, tag.name);
}
```

### Filtering Tags by Query

```ts
const filteredTags = await client.uploadTags.list({
  filter: { query: "hero" },
});
```

### Paginating Through All Tags

```ts
for await (const tag of client.uploadTags.listPagedIterator()) {
  console.log(tag.name);
}
```

### Creating a Manual Tag

```ts
const newTag = await client.uploadTags.create({ name: "hero-images" });

console.log("Created tag:", newTag.id, newTag.name);
```

**Important:** To assign tags to uploads, use `client.uploads.update()` or `client.uploads.bulkTag()` -- see `references/uploads.md` for details.

## Upload Smart Tags (Auto-Generated)

Smart tags are automatically generated by DatoCMS AI analysis of uploaded media. They are read-only -- you cannot create, update, or delete them.

### Listing Smart Tags

```ts
const smartTags = await client.uploadSmartTags.list();

for (const smartTag of smartTags) {
  console.log(smartTag.id, smartTag.name);
}
```

### Filtering Smart Tags by Query

```ts
const filteredSmartTags = await client.uploadSmartTags.list({
  filter: { query: "landscape" },
});
```

### Paginating Through All Smart Tags

```ts
for await (const smartTag of client.uploadSmartTags.listPagedIterator()) {
  console.log(smartTag.name);
}
```

Use cases for smart tags include auditing AI-generated tags, building tag clouds, and discovering asset categories across your media library.

### `UploadTrack` (response)

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Unique track identifier |
| `type` | `"subtitles" \| "audio"` | The type of track (audio or subtitles) |
| `name` | `string` | The human-readable name of the track |
| `language_code` | `string` | A valid BCP 47 specification compliant language code |
| `error` | `null \| string` | When status is `"errored"`, explains the reason for the error |
| `status` | `"preparing" \| "ready" \| "errored"` | The status of the asset |
| `closed_captions` | `null \| boolean` | Indicates if the track provides subtitles for the Deaf or Hard-of-hearing (SDH) |

## Complete Example: Add Subtitles and Organize Tags

```ts
import { buildClient, ApiError } from "@datocms/cma-client-node";

const client = buildClient({
  apiToken: process.env.DATOCMS_API_TOKEN!,
});

async function addSubtitlesAndOrganizeTags() {
  // The ID of an existing video upload in your DatoCMS project
  const videoUploadId = "existing-video-upload-id";

  // -------------------------------------------------------
  // Step 1: Add English subtitle track to the video
  // -------------------------------------------------------

  console.log("Adding English subtitle track...");

  const englishSubtitleTrack = await client.uploadTracks.create(
    videoUploadId,
    {
      type: "subtitles",
      url_or_upload_request_id:
        "https://example.com/subtitles/en.vtt",
      name: "English Subtitles",
      language_code: "en",
      closed_captions: false,
    },
  );

  console.log(
    "English track created:",
    englishSubtitleTrack.id,
    "Status:",
    englishSubtitleTrack.status,
  );

  // -------------------------------------------------------
  // Step 2: Add French subtitle track to the video
  // -------------------------------------------------------

  console.log("Adding French subtitle track...");

  const frenchSubtitleTrack = await client.uploadTracks.create(
    videoUploadId,
    {
      type: "subtitles",
      url_or_upload_request_id:
        "https://example.com/subtitles/fr.vtt",
      name: "Sous-titres Francais",
      language_code: "fr",
      closed_captions: false,
    },
  );

  console.log(
    "French track created:",
    frenchSubtitleTrack.id,
    "Status:",
    frenchSubtitleTrack.status,
  );

  // -------------------------------------------------------
  // Step 3: List all tracks on the video to verify
  // -------------------------------------------------------

  console.log("\nAll tracks on video:");

  const allTracks = await client.uploadTracks.list(videoUploadId);

  for (const track of allTracks) {
    console.log(
      ` - ${track.name} (${track.language_code}) [${track.status}]`,
    );
  }

  // -------------------------------------------------------
  // Step 4: Delete a track (e.g., remove the French one)
  // -------------------------------------------------------

  console.log("\nRemoving French subtitle track...");

  const deletedTrack = await client.uploadTracks.destroy(
    videoUploadId,
    frenchSubtitleTrack.id,
  );

  console.log("Deleted track:", deletedTrack.name);

  // -------------------------------------------------------
  // Step 5: Create manual tags for organizing assets
  // -------------------------------------------------------

  console.log("\nCreating manual tags...");

  const heroTag = await client.uploadTags.create({
    name: "hero-videos",
  });

  const tutorialTag = await client.uploadTags.create({
    name: "tutorials",
  });

  console.log("Created tags:", heroTag.name, tutorialTag.name);

  // -------------------------------------------------------
  // Step 6: List all manual tags
  // -------------------------------------------------------

  console.log("\nAll manual tags:");

  for await (const tag of client.uploadTags.listPagedIterator()) {
    console.log(` - ${tag.name}`);
  }

  // -------------------------------------------------------
  // Step 7: List all AI-generated smart tags
  // -------------------------------------------------------

  console.log("\nAll smart tags (AI-generated):");

  for await (const smartTag of client.uploadSmartTags.listPagedIterator()) {
    console.log(` - ${smartTag.name}`);
  }

  console.log("\nDone! Subtitles added and tags organized.");
}

addSubtitlesAndOrganizeTags().catch((error) => {
  if (error instanceof ApiError) {
    console.error("API Error:", error.response.status, error.errors);
  } else {
    console.error(error);
  }
  process.exit(1);
});
```
