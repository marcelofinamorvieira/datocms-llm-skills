# Trigger Eval Results

All checked-in eval outputs now live under this single root.

- root files here: the canonical latest accepted result snapshot.
- `historicalRuns/`: preserved published snapshots kept for reference.
- `adHocRuns/`: exploratory reruns, comparisons, and one-off checks.

Use the root files here when you want the repo's latest accepted numbers.
Use `historicalRuns/` when you need an older published snapshot.
Use `adHocRuns/` when iterating locally and comparing alternative runs.
